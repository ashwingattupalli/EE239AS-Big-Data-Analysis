## Usage ##
       
Usage: python <main-scripts>

<main-scripts>: q1_statistics, q2_linear_regression, q3_regression_model, q4_cross_validation, q4_superbowl, q5_testing_data, q6_twitter
                     
## Examples ##
If you want to run Question 1 (Calculate Statistics & Histogram),
$ cd Scripts 
$ python q1_statistics.py

If you want to run Question 6 (Twitter Ad-Celeb Week),
$ cd Scripts
$ python q6_twitter.py

## Looking at visualizations generated by Question 6
Run the command on terminal : pushd ./<path to folder>/Graphs/Question 6/; sudo python -m SimpleHTTPServer 8888
Once the server is up navigate on these links:
1. http://localhost:8888/visualize_ads.html
2. http://localhost:8888/visualize_celeb.html
3. http://localhost:8888/visualize_topics.html

## Description ##
- Import the entire zip file containing all the folders (Dataset, Graphs, Scripts) into an IDE (e.g PyCharm).
- Run <script>.py corresponding to the question you want to run as specified above.
- Graphs are displayed as a separate window.
- Output of each question is displayed in the console output.

## Folder Structure ##
- Dataset : Please add folders test_data and tweet_data in here.
- Graphs : Comprises of all graphs that are plotted for all questions.
- Scripts : Contains all the python scripts required to execute the project.
           - q1_statistics : Question 1
           - q2_linear_regression : Question 2
           - q3_regression_model : Question 3
           - q4_cross_validation : Question 4 - Part a.
           - q4_superbowl : Question 4 - Part b.
           - q5_testing_data : Question 5
           - q6_twitter : Question 6
           - topic_modelling, advertisement, celebrities : Question 6 utility files
